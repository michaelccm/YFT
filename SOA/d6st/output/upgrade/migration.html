Extension file sorting service Passed This service sorts the groups of the Elements (like TcClass, TcStandardType, etc.) defined in the BMIDE extension files. This will help the user in easily identifying the changes done with a file diff tool.
Sorting of elements in the extension files is successful.
Show details
Until now the elements were just grouped, e.g. all the TcClass elements would be together then TcStandardType and so on. In these groups, however, there was no ordering for the elements that were written, so modifying just an element used to create a lot of shuffling in the text in the extension file. This used to result in a lot of issues at the time of validating the changes done to the extension files. (The diff tools would return a lot of changes just from re-ordering of the lines.) After running this service the extension files will still have the elements written in groups, but the elements within each group will be sorted. With elements now sorted the changes done to the extension file will not shuffle all the contents, and the user will be able understand the diff generated. This service modifies the following files:
